## 📌 Sobre o Projeto

Aplicação web de cadastro de usuários desenvolvida com **Python + Flask**, utilizando integração com **MySQL** para persistência de dados e interface responsiva construída com **HTML, CSS e Bootstrap**.

Projeto criado com foco em demonstrar habilidades de desenvolvimento **Full Stack**, incluindo front-end, back-end e banco de dados relacional.

---

## ✨ Funcionalidades

* ✅ Cadastro de usuários
* ✅ Persistência de dados em MySQL
* ✅ Validação de formulário
* ✅ Layout Responsivo
* ✅ Estrutura organizada com Flask Templates
* ✅ Separação de arquivos estáticos

---

## 🛠 Stack Utilizada

| Tecnologia | Descrição                       |
| ---------- | ------------------------------- |
| Python     | Linguagem principal do back-end |
| Flask      | Framework web                   |
| MySQL      | Banco de dados relacional       |
| HTML/CSS   | Estrutura e estilização         |
| Bootstrap  | Responsividade/UI               |
